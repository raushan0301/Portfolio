## Summary

**Problem**

• Scattered media storage across platforms  
• Slow file upload and retrieval  
• Poor integration between database and files

**Solution**

• Cloudinary SDK for automatic optimization  
• Centralized media management system  
• Secure file upload with validation

**Why it matters**

• 40% reduction in administrative overhead  
• Automatic optimization of all uploaded media

## Engineering

**System Design**

Frontend  
• React with real-time upload feedback  
• Responsive design for mobile/desktop

Backend  
• Node.js + Express RESTful API  
• PostgreSQL for relational data  
• Cloudinary SDK for media storage

**Key Engineering Decisions**

• Cloudinary integration for automatic optimization  
• Role-based permissions with JWT auth  
• Secure file upload validation  
• Optimized database queries

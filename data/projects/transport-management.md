## Summary

**Problem**

• Manual, paper-based vehicle booking causing delays  
• No real-time visibility into vehicle availability  
• Lack of a transparent, hierarchical approval process

**Solution**

• Automated multi-role approval workflow system  
• Real-time dashboard for request tracking and tracking  
• Centralized system for vehicle pool management

**Why it matters**

• Digitized legacy campus logistics for better efficiency  
• Reduced approval time from days to just minutes  
• Improved accountability and data transparency

---

## Engineering

**System Design**

Frontend  
• React and Tailwind CSS for role-based admin dashboards  
• Real-time state management for live status tracking  
• Responsive interface for mobile and desktop users

Backend  
• Supabase for serverless database and authentication  
• PostgreSQL Row-Level Security (RLS) for data privacy  
• Custom workflow engine for multi-stage approvals

**Key Engineering Decisions**

• Rigid state-driven logic for approval routing  
• Automated triggers for real-time user notifications  
• Optimized database schema for complex role relationships

## Impact

**Impact Metrics**

• 100% digital transition from paper-based workflows  
• 70% reduction in administrative effort and overhead  
• Full transparency across 4+ administrative roles  
• Scalable foundation for future ERP module integration
